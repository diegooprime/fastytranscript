## ğŸ“– Usage

### Basic Usage (Zero Typing!)
1. Watch a YouTube video in your browser
2. Open Raycast and run "Fetch YouTube Transcript"
3. Press Enter
4. Transcript is instantly in your clipboard as Markdown!

### With URL
1. Open Raycast
2. Run "Fetch YouTube Transcript"
3. Paste YouTube URL (e.g., `https://www.youtube.com/watch?v=dQw4w9WgXcQ`)
4. Transcript copied to clipboard

### Extract Specific Segment
1. Run "Fetch YouTube Transcript"
2. Enter YouTube URL
3. Set start time: `1:30` (or `90` seconds)
4. Set end time: `5:00` (or `300` seconds)
5. Get only that segment!

### "No YouTube URL found"
- Make sure you have a YouTube URL in your clipboard OR
- Have a YouTube video open in Safari/Chrome/Arc/Brave OR
- Manually provide the URL

### "No transcript available"
- Video doesn't have captions in the selected language

- Some videos don't have any captions
